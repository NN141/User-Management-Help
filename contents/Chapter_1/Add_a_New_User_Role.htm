Add a New User Role

  1. Go to Settings   Manage  User Roles.
  2. Click the Add icon  .Add User Role window will appear.
  3. In the User Role box, type the name of the User Role. (E.g. FM Manager)
  4. User Rights Configuration  
    1. To view User Rights available under a typical preconfigured User Role Group, expand by clicking the respective  icon.
    2. To add all the User Rights grouped under a typical User Role to the new User Role at once, select the checkbox adjacent to the typical User Role.
    3. If not, select each checkbox, to add User Rights individually to the User Role as you require.   E.g. User Role - e.g. Administrator User Rights – Administrator User Rights (If the “Administrator User Rights” checkbox is selected, all the User rights under “Administrator User Rights” will be applied. Or else, select individual User Rights considering the permissions you need to provide to each functionality in the App)
  5. Click Add.
