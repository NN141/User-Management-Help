Edit an Existing User Role

  1. On the User Roles page, find the User Role that you want to edit and click the respective Edit icon  .
  2. To search User Right(s) by its name, type in User Right or part of it in the Search box. User Rights will be filtered according to the typed text.
  3. Select the required search criterion in the drop down box to filter the User Rights based on the following criteria.  

    All Items

    Use to view all the User Rights

    Selected Items

    Use to view only the selected (ticked) User Rights

    Unselected Items

    Use to view only the User Rights that are not selected
  4. Select/deselect User Rights for the respective User Role as required and click Update.
