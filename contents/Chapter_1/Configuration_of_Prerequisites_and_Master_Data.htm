Chapter 1

Configuration of Prerequisities and Master Data

You need to configure the following prerequisites and master data, before registering Users in User App:

 

Step 1: Prerequisites Configuration

  • User Roles
  • App Configuration

Step 2: Master Data Configuration

 

After the User App is ready with initial configurations (described under Step 1), you need to configure the following master data.

 

  • User Groups
  • User Types
    • Default User Types
    • Custom User Types
  • User Message Templates
