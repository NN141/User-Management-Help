User Types

User Type is a category that is used to group users according to the job or task they perform. ‘User Type’ defines a class of users such as Technician, Supervisor, Customer, Operator etc..


User Types are divided into two groups as follows:

  • Default User Types
  • Custom User Types
1.   Default User Types

 

Default User Type is a group that categorizes the user types that are configured under respective Apps. User Types that are configured under other Apps will be displayed under User Types Default User Types in the User App, e.g. User Type ‘Staff’ comes from the Organization App.

 

Note –

  • Default User Types cannot be configured/edited or deleted in User App.
  • If a User’s User Type is a Default User Type (e.g. Staff), you will not be allowed to change the User Type in the User Details page.

View Default User Types

  1. Click the Settings icon   on the Home page and select User Types under Configure.
  2. User Types page will appear. Default User Types will be displayed in the first section of the page. Existing Custom User Types that are configured in the User App will be displayed as well.
2. Custom User Types

 

Custom User Type is a group that categorizes the User Types that are configured by the users in User App.

 

Note – You will be able to change the User Type of a User in User Detail page, if the existing User Type is a Custom User Type only.

 

This section describes the following;

  • Configure a Custom User Type
  • Edit a Custom User Type
  • Delete a Custom User Type

Configure a Custom User Type

To configure a Custom User Type;

  1. Click the Settings icon  on the Home page and select User Types under Configure.User Types page will appear.
  2. Scroll down and go to the Custom User Types section.
  3. Click Add New Custom User Type link.
  4. In the User Type box, type the required User Type, e.g. Call Center Operator, Engineer.
  5. Click Add. Click Cancel to abort the operation.

Edit a Custom User Type

To edit a Custom User Type;

  1. Click the Settings icon  on the Home page and select User Types under Configure. User Types page will appear.
  2. Scroll down and go to the Custom User Types section.
  3. Hover over the required Custom User Type that you want to edit.
  4. Click the Edit icon  that appears.
  5. Edit as required and click Save.

Delete a Custom User Type

To delete a Custom User Type;

  1. Click the Settings icon  on the Home page and select User Types under Configure. User Types page will appear.
  2. Scroll down and go to the Custom User Types section.
  3. Click the respective Delete icon   to the right of the Custom User Type that you want to delete.
  4. A message appears to confirm the deletion. Click Delete to confirm. To abort the operation, click  of the message box.
