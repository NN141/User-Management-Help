User Message Templates

This configuration is used to create message templates and that can be used whenever a new message or notification with respect to a User registration/ password management needs to be sent.

The templates will hold the common parts or sections of the messages that need not have to be written repeatedly.

This section describes the following:

  • Configure a New Message Template
  • Edit a Message Template
  • Delete a Message Template

 

1.   Configure a New Message Template
  1. Click the Settings icon  on the application home page and select the User Message Templates link under the Configure section. A list of existing templates will be displayed.
  2. Click on the Add New Message Template link to the top right corner. New Message Template page will open.
  3. Type Name, Subject, and Message as required for the Message. Use the Show Placeholders link to add system generated values to the Message text.
2.   Edit a Message Template
  1. Click the Settings icon on the application home page and select the User Message Templates link under the Configure section. A list of existing templates will be displayed.
  2. Search for the required template and click on template name to go to Template Detail page.
  3. In the Notification Template tab, edit message Name, Subject, Recipients as required.
  4. In the Message tab, edit the message text as required.
  5. Click the Rich Message tab and add additional formatting to the message, e.g., font color, size etc.
  6. Click the Short Message tab and add a short message to be sent to mobiles.
  7. Click the Attachments tab to include attachments in the message.
    1. Click Add icon  on the top right corner. New Attachment window will appear.
    2. Type Name of the attachment.
    3. Select Type of the attachment.
    4. Add Options if required.
    5. Click the marked area to browse and add the attached fil Or drag and drop the file to the area marked as ‘Drag your file here’.
    6. Click Upload.
    7. Attachment will be listed under the Attachments section.
    8. Click Edit icon to edit the required attachment or click the respective Delete icon  to remove an attachment.
3.   Delete a Message Template
  1. Click the Settings icon  on the application home page and select the User Message Templates link under the Configure section. A list of existing templates will be displayed.
  2. Search for the required template and click on template name to go to Template Detail page.
  3. Click the Delete this Template link or  icon in the right side panel.Confirmation message will appear. Click Delete  to continue or click  to abort.
  4. If the Message Template is in use, message box will appear stating that ‘This message template is in use and cannot be deleted’.
