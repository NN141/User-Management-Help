Hidden Users

Hide a User

There shall be users that are not required in the current active User list. If these users are already in use, the system will not allow you to delete the same.

 

Therefore these users can be hidden the way that the same are removed from the current active user list.

  1. Click the Settings icon  and select Users under View section. User Search page will appear.
  2. Search for the required User and click the User ID to go to respective User Detail page.
  3. On the Side Bar to the right side, click the Delete this user link or click the Delete icon. A message will pop up stating that the respective user is in use and cannot be deleted.
  4. To hide the user, click Hide this User instead.
  5. After the user is hidden a banner will be displayed on the right Side Bar of the respective User Details page stating that the user is currently hidden.
  6. The hidden user will be added to “Hidden Users” list which can be viewed at SettingsàManage àHidden Users.

View & Unhide Hidden Users

Hidden users are the users that are already in use, but removed from the current active User list.

To view the hidden users and to unhide a Hidden user:

  1. Click the Settings icon and select Hidden Users under Manage section. Hidden User Search page will appear.
  2. Select the required Hidden user that you need to make visible and click the User ID. Selected hidden user’s detailed page will be opened.
  3. On the Side bar, click the Make visible link to make the selected hidden user visible in the current active user list.
