Edit User Details

  1. Click the Settings icon  on the Home page and select Users under View section. User Search page will appear.
  2. Search for the required User and click the User ID to go to respective User Detail page.
  3. Click the User Details tab.
  4. To edit user details, hover over User Details and click the Edit icon  that appears. Edit mode will open.
  5. Edit the details of user as required.
  6. Click Save.
  7. Click the User Roles tab to view the roles assigned for the user.

Edit User Profile Picture

  1. Click the Settings icon  on the Home page and select Users under View section. User Search page will appear.
  2. Search for the required User and click the User ID to go to respective User Detail page.
  3. Click the User Details tab.
  4. On the Side Bar, under the profile picture, click the Edit link.
    1. Click upload area and select the relevant User’s image from browsing or drag the required image to marked are Image will be uploaded.
    2. Alternatively, you can click the Take a picture link to capture an image through the web camera and upload.
    3. If you want to close the profile picture’s Edit mode, click the Cancel link.
  5. To remove the current image, go to Edit mode and click Remove this picture link.
