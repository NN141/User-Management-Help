Delete a User

Back to Operations Related to Registered Users Diagram

 

To delete a user:

  1. Click the Settings icon  on the Home page and select Users under View section. User Search page will appear.
  2. Search for the required User and click the User ID to go to respective User Detail page.
  3. Click the User Details tab.
  4. On the Side Bar to the right side, click the Delete this User link or click the Delete icon .
  5. Confirmation message will appear. Click Delete  to confirm deletion.
  6. If the user is in use by the system, a message will appear stating that the user cannot be deleted. Instead you can hide the user from active list by clicking Hide this User instead.
