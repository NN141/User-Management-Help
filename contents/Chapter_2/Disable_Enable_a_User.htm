Disable/Enable a User

 
Back to Operations Related to Registered Users Diagram

Disable this User link on Side bar of User detail page allows you to disable the respective user’s login to the system. Generally this function will be done by an administrator or an authorized person.

 

If a user is disabled, the respective user will not be able to login to the system, but will be able to check-in to the respective building as a visitor in Visitor app.

 

  1. Go to Settings menu àView àUsers
  2. Go to the detail page of the required User.

Click the Disable this User link on the side bar of User Details tab. The message will pop up stating that the User is disabled and the text of the link will be changed to ‘Enable this User’.


To enable a disabled User, click the Enable this User link on the side bar of User Details page.
