User Search (View User Details)

Back to Operations Related to Registered Users Diagram

User Search is used to search and view existing user details. This helps to find the required users, to edit and update their details.

To go to User Search:

 

Tip – On the Quick Info window, click the Pin  icon to add the User to               the    Pin Board.

      • On the Quick Info window, click the User name to view the respective           User’s details page.

  1. Click the Settings icon  on the Home page and select Users under View section. Or click the Users tab on the Home page. User Search page will appear.
  2. Use the given filters to search users with given parameters or type the required User ID in the Search Users box. Matching records will be filtered.
  3. Once you get the search results, click the Quick Info icon  to view important information of the respective user on the quick information window.
  4. Click the required User name to go to its detailed page.
    • Click User Details tab to view the details of the selected user.
    • Click the Roles tab to view the User roles that have been assigned through the selected User Group.
  5. You shall find some tabs that differ according to the User Type (e.g. User Type: Tenant User  à Tenant User Details tab) and enabled other tabs, e.g. If the Seat Booking App is enabled, Seat Details tab will be displayed.

Tabs in User Details Page

User Details Tab:

 

This tab shows the important details and user’s personal details.

 

User Roles Tab:

 

This tab shows the User Roles that are tied with the respective User.

  

Preferences Tab:

 

This tab shows the User Roles that are tied with the respective User.

  

Access Card Details Tab:

 

This tab appears when Access Card App is enabled and allows Access Card assignment operations for the user.

  

Seat Details Tab:

 

This tab appears when Space Management App is enabled and lists given Seat Allocations for the user.

  

UMS Dashboard Settings Tab:


UMS Dashboard tab is displayed, only if the UMS App is enabled.

This feature (on the detailed page of each User) allows you to configure the Map settings of individual Users, which will be reflected on the respective User’s UMS dashboard.

If you configure UMS Dashboard settings on User Detail page, those settings will override the following and will be applied as the UMS Dashboard settings of the respective User.

The settings configured on the UMS Dashboard Settings tab of the User Detail page will override;

  • The respective User’s User Group UMS Dashboard Settings
  • Default Map Settings under App Configuration in UMS App

The Map Settings along with the Location configured here will be taken as the Location/Map settings of the respective user on UMS App Dashboard.

Preferred Locations: This section is used to add the Locations of which energy consumption is required to be monitored on UMS App Dashboard of the respective User. Respective User will be able to monitor the energy consumption of only the selected Locations under Preferred Locations section.

Tip –Only the Locations tied with Meter Groups in UMS App will be listed down in the Location box on the Add Preferred Locations window..

 

Visits Tab:

 

This tab appears when Visitor Management App is enabled and lists registered Visits by the given user.

 

Staff Tab:

 

If the User is created using the Organization App as a Staff user, this tab will appear.

 

Induction Details Tab:

 

This tab appears when Induction App is enabled and allows Induction conducted for the user.
