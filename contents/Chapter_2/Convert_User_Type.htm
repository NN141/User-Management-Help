Convert User Type

Back to Operations Related to Registered Users Diagram

 

Convert User Type link allows to convert the User Type of the respective User to another User Type.


E.g. from ‘Staff’ User Type to ‘Tenant’ User Type
 

  1. Click the Settings icon  on the Home page and select Users under View section. User Search page will appear.
  2. Search for the required User to convert and click the User ID to go to respective User Detail page.
  3. Click the User Details tab.
  4. On the Side bar, under Related Items, click the Convert User Type link. User Type conversion window will open.
  5. Selected user’s User ID and the current User Type will be displayed.
  6. In the Convert To box, click and select the required new user type to convert to from the available user type list.
  7. Click Proceed to continue conversion.
  8. Converted user registration page will open. Add necessary new detail and proceed.
