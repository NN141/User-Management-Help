Blacklisted Users

Back to Operations Related to Registered Users Diagram

If you need to prevent a person (User in the system/ Visitor/any other person who is not currently registered in the system) from interacting with your organization due to any reason, you shall add the respective person to the Blacklist or create the third party person as a Blacklisted User.

 

Blacklisted Users are not allowed to interact with the system.

 

e.g. check-in to the respective premises via Visitor App, book facilities via Facility Booking App etc.

 

Identification Number

 

Identification Number is mandatory in Blacklisting Users and it should be a unique value in the system.
The system validates and filters Users using their Identification Numbers, when blocking the users from activities in the system such as;

  • checking-in to the respective premises via Visitor App
  • booking facilities via Facility Booking App

Note: iviva system will not allow you to upload/import user data associated with Identification Numbers of Blacklisted Users through Bulk Data Upload function in the following apps;

  • User App – User upload
  • Property Management App – Landlord User / Tenant User upload
  • Facility Management App – Vendor User Import
  • Visitor App – Visitor Import
  • Organization App – Staff Import
Add Blacklisted Users

 

User App provides 2 options for you to add a Person to Blacklist;

  1. Add a selected User to Blacklist on User Detail Page
  2. Add a new Blacklisted person on Blacklisted User Search Page

Note: Visitor app allows to add Visitors to the Blacklist on Visitor detail page.

Tip – Once a User is blacklisted, the User will be shown under Blacklisted Users and Hidden Users in Manage section. (Blacklisted Users’ records will be hidden and will not be shown under the active user list on User Search page. In the detail page of a Blacklisted User, only the ‘Blacklisted User Details” tab will be shown.

 

Add Selected Users to Blacklist on User Details Page

This provides a link on a User’s user detail page to add the selected to the Blacklist.

  1. Go to Settings menu àView àUsers
  2. Go to the detail page of the required User.
  3. Click the Add User to Blacklist link on the side bar of User Details tab. A window will appear to enter the reason for blacklisting.
  4. In the box named Reason (Blacklisted Reason), type the reason for blacklisting the selected User.
  5. Click Add. A message will be popped up confirming that the user has been added to the Blacklist and you will be redirected to the Blacklisted Users search page.

 

Tip – In case the respective User’s Identification Number has not been entered yet, a message will pop up asking you to update the Identification Number.  

• Click the Edit icon on the User Details tab and Enter the respective Identification Number and click Save.

 

This situation is possible only if you enable the User blacklisting function, after Users are registered in the system without Identification Numbers.

 

Add a new Blacklisted Person on Blacklisted User Search Page

This method can be used to add persons, (those who are not yet registered in the System) to the Blacklist.

  1. Go to Settings àManage à Blacklisted Users.
  2. Click the Add New Blacklisted Users link on the side bar of Blacklisted Users search page.
  3. In the Name box, enter the name of the User.
  4. In the Identification Type box, enter the type of the Identification Number used to identify the Blacklisted User.

E.g. (1) Identification Type: Passport

           Identification Number : <Passport no>

       (2)Identification Type: National Identity Card (NIC)

             Identification Number : < NIC No>

  1. In the Identification Number box, enter the Identification Number of the User which is a unique number.


Identification Number: This Identification No. is used to validate and identify the Blacklisted Users in Apps such as Organization, Visitor, FM, and Property etc. and in Bulk Uploading to prevent Blacklisted users utilizing the activities related these Apps.


Identification Number E.g.: <Passport Number>

 

  1. In the Reason box, enter the reason for blacklisting the respective User (Blacklisted Reason).
Add Extra Search Criteria to Blacklisted User

 

Extra information can be added to Blacklisted Users, which can be used as search criteria to identify blacklisted users using several Identification Numbers. This helps to have a thorough search and identify the Blacklisted Users, using the other Identification numbers in order to prevent them interacting with the system.

e.g. Passport Number, License Number


To add extra search criteria;

  1. Go to Settings àManage à Blacklisted Users.
  2. Click the name of the Blacklisted User you require to go to its detailed page.
  3. Under the Extra Search Criteria, click the Add icon. A window to add Identification Type and Identification Number which can be used as Extra search criteria.
  4. In the Identification Type box, enter the type of the Identification number.
  5. In the Identification Number box, enter the Identification Number belongs to the Identification Type.
  6. Click Add.
  7. Click Add icon and add more Identification numbers if required.
Remove a Blacklisted User from Blacklist

 

If you need to remove a user who is already marked as a Blacklisted User, from the Blacklist;

  1. Go to Settings àManage à Blacklisted Users.
  2. Click the name of the required Blacklisted User to go to its detailed page.
  3. Click the Remove User from Blacklist link. The respective User will be removed from the Blacklist and will be added to the User list again.

Tip –Once a User is removed from the Blacklist;

        • The User will be added to the active User list and will be shown in               User Search page

        • All the related data of the User will be shown under each respective             tab in User detail page.

 

Tip –Once a User is removed from the Blacklist;

        • All the related data of the User will be shown under each respective            tab in User detail page.

       • The User will be added to the active User list and will be shown in              User Search page.
