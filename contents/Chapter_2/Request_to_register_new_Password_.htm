If a user needs to change the login password, a request has to be made using the Request to register new Password link given on the side bar of User Details tab on User detail page.


To request for a password change;

  1. Click the Settings icon  on the Home page and select Users under View section. User Search page will appear.
  2. Search for the required User and click the User ID to go to respective User Detail page.
  3. Click the User Details tab.
  4. On the Side bar, under Related Items, click the Request to register new Password link. A message box will appear to confirm the operation.
  5. Click OK. An email notification will be sent to the respective user containing the instructions to register a new password.

 Setting under App Configuration

 

A notification template can be selected under App Configuration, which can be applied to this email notification. If not configured, the default password change screen will appear to register the new password.
