Reset User Password

Back to Operations Related to Registered Users Diagram

 

If you need to reset the current password of a user and replace with a system generated password, it can be done by using the Reset Password function given on the side bar of the User Details tab of User detail page.

 

To reset a password of a user in the system:

  1. Click the Settings icon  on the Home page and select Users under View section. User Search page will appear.
  2. Search for the required User and click the User ID to go to respective User Detail page.
  3. Click the User Details tab.
  4. On the Side bar, under Related Items, click the Reset Password link.
  5. On the Reset Password window, click Generate a new password. A new password will be generated and it will be displayed on the screen.

 Setting under App Configuration

A user is allowed to change the password ONLY if “Allow User to Manage Password” is checked on the Login Details tab on the New User Configuration wizard (Settings à Register à New User).
