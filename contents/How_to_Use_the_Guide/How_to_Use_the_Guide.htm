How to Use the Guide

 

This guide is mostly designed for Administrators who perform operations related to User Management.


In this guide, the User Management Application Process is divided into 6 Steps. The steps and the main titles under each Step are shown in the User Management App Process Diagram. The steps are sequentially described under chapters in the Guide. You will find the Steps described under chapters as stated below.

 

Chapter Step
1

Step 1 : Configuration of Prerequisites
Step 2 : Master Data Configuration
2

Step 3 : User Registration
Step 4 : Operations Related to Registered Users
Step 5 : Bulk Data Import
3

Step 6 : Reports

 

Conventions Used in the Guide 

The following shows the description of icons and conventions used throughout the guide.

 

 

Note – A note provides important information that you should know in addition to the standard details.

 

Tip – This represents any tips regarding the topic/steps that have been discussed.

 

Attention – Represents any warning before proceeding with any operations.
