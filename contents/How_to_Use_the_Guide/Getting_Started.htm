How to Access the “User Management” App

 

To access the User Management application;

 

 

Login to iviva. Your iviva home dashboard will appear.

Click the User icon  on the Application Menu on the left. User Management App’s home page will appear.

 

Application’s Home Page

 

User Management application’s home page consists of the following:

 

Features on Home Page Description

Tabs  Dashboard

Dashboard is displayed under the first tab of the Home page.

This shows various weblets related to the User Management application.

Tabs  Users

Users tab shows the users that are registered in the application and provides User Search to search for the required users.

The same User Search function is found under Settings icon.

Users tab provides quick access to Users search.

New User button provides quick access to the new User registration page. 

 Settings Icon

Settings icon on the Home page gives you access to settings and configuration items within the User Management app.

 

Dashboard

Dashboard shows various weblets related to the User Management application.On User Management Dashboard the following weblets can be found.

 

User Types with Registered Users:This shows User Types (only the User Types with registered users) along with their total number of users that are registered in the system.

 

My Profile: This shows the basic and important details of the logged on User, such as User Type, Site, Phone, E-mail, along with the User’s image.

 

View Weblet Details

User Types with Registered Users

Click the Search icon  next to the User count of a particular User Type to view the registered users of the respective user type.

My Profile Weblet

This weblet shows the User Type, Site, Phone, E-mail, and User’s profile image of the logged on User.

 

If the Seat Booking app is enabled, User’s Permanents Seat and Temporary Seat locations will be displayed as well.

 

To edit the User Profile, click the user’s profile image.

 

 

Tip –  On the Registered User Types in the system weblet, to view the rest of the User types in the list that are not visible on the weblet, hover over the top or bottom of the User Type list and click the Up arrow  or Down arrow. 

 

System Toolbar

System toolbar is displayed on the top right corner of the screen. It has the following icons:

 

Icon Description

 Message Inbox

Clicking the ‘Message Inbox’ icon will open a page with a Message List.

Click a message title on the left pane to view the details of the selected message.

Click the Inbox link on the left pane, to open the Message Search box.

 

Inbox Message Search Window :

  • Click the Message Inbox icon in the System Toolbar.
  • Click the Inbox link on the left pane. Message Search box will appear. By default, Inbox will be opened.
  • In the Search Messages box, type the required search text to filter the messages accordingly.
  • Click Sent After  and Sent Before icons and select relevant dates to filter messages according to the message sent date.
    • Sent After: This field allows to filter messages sent after the selected date.
    • Sent Before: This field allows to filter messages sent before the selected date.
    • Time: Click the clock icon to select the required time, to filter messages according to the message sent time
  • Show Hidden Messages: Click Show Hidden Messages check box to view messages that are already hidden.
  • Click message title to go to the respective message’s detailed view.
  • In the detailed view of message, click the Hide this message checkbox to hide the selected message from the normal view.

Tip –  Click the  in the Date or Time input boxes, to remove the entered date or time respectively from the boxes.

 Pinboard

Pin Board icon shows the items pinned by the user.
 Help Icon

Help icon opens the help files
 Edit Main Dashboard
  • Clicking on Edit Main Dashboard icon allows you to drag the respective App icon and rearrange it.
  • To select the required weblets onto the dashboard of respective App’s homepage;
    • Click the User profile on the top left corner and select ‘Home’ to go to the system Dashboard.
    • Click the  icon. Weblets for all the Apps will be displayed by default.
    • To select only the weblets of a particular App, select the name of the App from the drop down box.
    • Select the required Weblet/weblets and click the respective Add to dashboard button. The respective weblet/weblets will be added to the dashboard of the respective App(s).

 

How to Log Out from the System

 

To log out from the system;

 

  1. Click the user profile on the top left corner, next to your User ID.
  2. Click Log Out menu item on the menu. Current User will be logged out from iviva and the User Login page will appear.
