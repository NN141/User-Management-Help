View User Role Report


User Role report shows User Roles that have configured to access each Application in the system.


To view User Role Report:

  1. Click the Settings icon  and select User Role Report under Reports. User Roles for Applications in the system will appear.
  2. Select the required App Name to view the User Roles of the selected App.
  3. Select the required User Role to filter the applications that have the selected User Role.
  4. To export the report data to Microsoft Excel format, click Export to Excel link.
