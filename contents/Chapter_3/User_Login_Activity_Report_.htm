User Login Activity Report

This is used to create, generate and view reports on User Login Activity details.

You can view User Login Activities of a given User or of all Users within a given period of time in a selected Site.

To view User Role Report:

  1. Click the Settings icon and select User Login Activity Report under Reports. User Roles for Applications in the system will appear.
  2. Select the required Time Duration, User and the Site.
  3. To generate the created Report into PDF/Word/Excel/Excel Raw, select the required Format and click Generate.
