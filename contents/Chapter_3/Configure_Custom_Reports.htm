Chapter 2 : Reports

Configure Custom Reports

Authorized Users shall design custom reports in User Management Application.

To design a Custom Report;

  1. Click the Settings icon  on the Home page and select Custom Reports under Configure. Custom Reports Search page will appear.
  2. Click the Design a new report link. Report configuration page will appear with three tabs on it namely Report Settings, Data Sources and Layout & Widget.
Report Settings
  1. Enter Report Settings:
    1. On the Report Settings tab, in the Report ID box type the name of the report.
    2. In the Date Format box, select the date format.
    3. In the Time Format box, select the relevant time format.
    4. Select the Currency Format and Time Zone.
Data Source
 
  1. On the Data Sources tab, click the Data Source box and select the required Data Source.
  2. To add a new Data Source:
    1. On the Data Source tab, click the Add icon.
    2. In the New Data Source window, in the Data Source box, type a name for data source.
    3. Select the Base Object. (e.g. If the report is on users, select Users as the Base Object). A Base Object contains its related Fields. (e.g. If the Base Object is Users, it will have its related fields such as User ID, User Type etc.)
    4. Click Add.

Fields

  1. Select the required Fields
    1. Under Fields click the Click to select a field link. A box with fields that can be added to the report will be displayed.
    2. Click on required fields to add.

Note – 

      • Select a Field:  The prefixes that appear in front of the fields                 denote the type of values accepted by the fields.  

        e.g. T : Text        
       N:  Numeric  D: Date/Time      

   • Depend on the report you want, you may need to add the same             Field more than once. e.g. To show User Type along with User Type       Count, you need to add User Type field twice (UserType, UserType1)       as the second field is used to calculate the User Type Count.


  1. Repeat the step 6 to add more fields.
  2. To delete an added field, click the respective icon .
  3. To preview the selected Data Source and its added Field, click .

Filter Conditions

  1. Add Filter Condition. (This is used to filter the records in the report using the field and filtering condition)
    1. Under Filter Conditions section, click the Add link.
    2. Select the Filtering Field from the list. (e.g. User ID)
    3. Select the filtering condition from the given box. (e.g. Exists)
    4. A third field will appear based on the filtering condition you select. (e.g. Equals). Then type the value which is required to satisfy your filter condition. (e.g. User Type  Equals ‘Technician’)
  1. Repeat the step 10 to add more filter conditions.
  2. To remove a filtering condition, click the respective  icon.

Grouping

  1. Grouping the data in the report:
    1. To group the data, select the Group Data check box. All the fields will appear.
    2. Click on the list box of the required field (e.g. UserType1) and select the field’s property (e.g. Count) that you want to group by.e.g. Field : UserType1  Group By : Count à This will show the count of each User Type.

Sorting

  1. Sorting the data in the report:
    1. To group the data, select the Group Data check box. All the fields will appear.
    2. If you want to sort the data in ascending order, select the Ascending check box.
Layout & Widget
 
  1. Specify Layout and Widget settings.

    If you want to split the report into sections by “User Type”, in the Section By list, select the field “User Type”. In the Section Name box, type the name of the section you want to display on the report.

    1. Click the Layout and Widget tab.
    2. In the Report Title box, type a relevant title for the report.
    3. In the Report Layout list, select the layout of the report you want. (e.g. Simple Layout, Two widgets). Add a Widget link appears.
    4. Under the Section 1, click the Add a Widget link. A screen to add the new widget appears.
    5. In the Widget list, select the widget you want. (e.g. Bar chart, Pie Chart)
    6. In the Data Source list, select the required data source for the selected widget.
    7. Click Add.  Depending on the selected widget, modify the default values displayed as the widget data, if and as required.
    8. Select the Split into Sections check box to split the report into sections based on field selected on the Data Source.
    9. Modify the default values for the Section By and Section Name fields, if required.

e.g. If you want to split the report into sections by “User Type”, in the Section By list, select the field “User Type”.

In the Section Name box, type the name of the section you want to display on the report.

Note – Report Layouts

Simple Layout: This can show only one type of widget (Table or Pie Chart or Bar Chart) in the report.

Two Widget Layout: This layout type can be used to display two types of widgets in the same report.

  1. If you have selected ‘Two widgets’ as the Report Layout, repeat the steps 15-d - 15-i for a widget in the Section 2.
  2. After specifying all the required settings, click Save to save the Report design configurations.

Delete a Widget

On the Layout & Widget tab, click the  icon. Delete this widget link appears.

  • Click the Delete this widget link.

Preview Designed report

To preview the designed report in the web browser with information,

  • Click the icon .

You can customize the report view by changing the properties in the design. The illustration shows the report properties explained below.

Label:

Label displays the name of the respective field in the report. You can edit the label accordingly.

Order:

Click Up arrow  once to move the respective field before the next field on top of it. Click the Down arrow  once to move the respective field one field below. You can click the Up or Down arrow multiple times according to the place that you want to display the respective field on the report. This helps to arrange the order of the fields that appear on report.

Show:

To select the fields that you want to appear on report, select the respective check boxes under Show column.

Format:

To change the format of the field as you want, select the required format by clicking the respective list box. By default it will show the default value. e.g. To show Incident Date Time field on the reports, change the format to Date. To display the Time along with the Date, select the Include Time check box.


Report Layout Formats

Format of the Field Description
Default This shows the default value from the database
Currency Used to display report data in currency format
Integer Used to display report data in integer format. This will help to calculate the Total of the integers in the same field.
Number Used to display numbers even with decimals.
Date Used to display a Date in report. Selecting the Include Time check box will display the Time along with the Date.
Custom This can be customized by the user according to the requirements.

Editing Custom Format

  • Click the respective box under Format column and select Custom. #{row.<FieldName>} is the field name of the underlying Data Source of the software application frame work hence it should not be changed. However you could add textual prefixes and suffixes which would be repeated along the data for each row.

e.g. Label: Web Address Format: <prefix>#{row.<yourWebURL>}<suffix>

Total:

This is used to display the total of all the numeric values/numbers in a row that belong to a selected Field.


Note –After doing the changes in the report layout and widget, preview the report to verify the outcome.
