View User Report

User Report shows a list of users that are registered in the system. User can be filtered based on the selected User Group and Site.


To view User Report:

  1. Click the Settings icon  and select User Report under Reports.
  2. On the User Report, select the required User Group.
  3. Select the require Site. Users will be filtered according to the selected User Group and Site.
  4. To view basic user information, click the respective Quick Info icon  . On the Quick Info window, click the User Name to go to the detailed page of the User. On the Quick Info window, click the pin icon to add the User to the pin board.
  5. To export the report data to Microsoft Excel format, click Export to Excel link.
