Edit Report Design

To edit a report that is already designed;

  1. On the User Management Home page, click the Settings icon and select Custom Reports under Configure. Reports that have been already designed will be displayed.
  2. To view the report,click the Report name that you want.
  3. Click the Edit icon .Report configuration page will be opened.
  4. Do all the required configuration changes.
  5. Click Save.
